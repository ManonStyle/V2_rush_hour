Solveur

Remise du solveur.

Vous devez remettre un exécutable qui puisse s'exécuter sur une machine "standard" du crémi (comme par exemple mcgonagall).

L'exécutable doit s'utiliser de la manière suivante :

./solveur <a|r> <filename>

Le premier paramètre indique le type de jeu à résoudre :

r pour "rush hour"

a pour ane rouge

Le second paramètre indique le nom d'un fichier contenant la configuration à résoudre (cf. format spécifié plus haut).

La sortie du programme doit être un entier qui indique le nombre optimal de mouvements pour résoudre le jeu, et -1 si le jeu n'est pas soluble.

Exemple :

>./solveur r config1.txt

13


Les solveurs qui ne respectent pas ces spécifications seront discalifiés.
